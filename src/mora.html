Mora Drand Design

A comprehensive design system created for Mora Drand, focusing on brand coherence and extensibility.

Overview

The Mora Drand project involved developing a scalable UI kit, colour palette, typography guidelines, and prototypes for key user flows.

  • Logo refresh and colour explorations
  • Component library (buttons, forms, cards, modals)
  • Accessibility compliance and responsive layouts
  • Documentation site to support handoff to developers

Role

Lead Designer – conducted user research, created visual assets, and collaborated with developers to implement the design system.
