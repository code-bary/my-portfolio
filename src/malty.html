Malty Website

An immersive multi-page website built for the Malty brand, with emphasis on storytelling and interaction.

Overview

Malty is a conceptual design project to explore clean, modern UI with rich animations and a flexible layout. The site includes:

  • Landing page with animated hero section
  • About page detailing brand values
  • Services page with interactive cards
  • Contact form with validation and micro-interactions

Role

UI/UX Designer – responsible for wireframes, high-fidelity designs, front-end implementation using HTML/CSS/JS, and accessibility testing.
