# my-portfolio

This repository holds the source for a simple designer portfolio website focused on branding and UI/UX work. The goal is to showcase a unique identity and clean, modern interactions while giving clients insight into process and deliverables.

## Structure

- `src/index.html` – main landing page
- `src/styles.css` – basic styling rules
- `assets/` – (create as needed) images, logos and screenshots of projects

## Getting started

You can preview the site with any static server. From the root of the project run:

```sh
cd src
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Designing a unique UI/UX portfolio

1. **Define your brand** – choose a colour palette, typography, and voice that reflect your style.
2. **Focus on clarity** – make navigation obvious, highlight key projects, and keep text concise.
3. **Tell a story** – for each case study explain the problem, your approach, and outcomes.
4. **Use interactive elements** – subtle animations, hover effects, and transitions can elevate the experience.
5. **Mobile first** – ensure the layout is responsive and touch‑friendly.
6. **Keep details** – add contact links, downloadable CV, client testimonials, and process descriptions.

Feel free to iterate on the markup and CSS to create a completely unique look; this template is just a starting point.
